    PJ-T42 型单路“可编程时控模块”是由多功能微电脑 IC 直接封装于 PCB 板上，内含 LCD 驱动电路，采用 LCD 液晶显示，具有以下功能：

一、 特点及说明

1. 实现每天 42 开 42 关（ 42 组）编程定时
每一组定时可设定星期、小时、分、开 / 关。
可随时进入任一组，将其资料取消使该组失去作用。
当需要时可随时将已取消的资料重新呼叫（恢复）出来，使该组发生作用。
智能键盘锁，防止非法修改程序。

2. 定时控制周期

日控循环或周控循环 ( 每周可实现每天不同定时设置 ) 。

3. 时钟采用 24 小时计时制

二、主要技术参数

1. 晶振频率： 32768Hz           7. 储存温度： -40C～±60C
2．工作电压： 3V                8. 工作温度： -20C～±60C
3. 静态电流： 7 ～ 9uA          9. 显示方式 : 中文 LCD
4. 高电平驱动电流： 2mA         10. 外型尺寸 : 57mm * 29mm
5. 低电平驱动电流： 2mA         11. 显示尺寸 : 17mm * 47mm
6. 计时误差 : ±0.5 秒 / 天     12. 时控范围 :1 分 ～ 168 小时

四． PJ-T42 可编程定时模块工作原理

    典型的电路如上图所示，模块 J1 端口输出的开关信号通过三极管 N1.N2 等元件构成的放大电路放大后驱动继电器 J1 ，在开启的状态时， J1 端口输出高电平，继电器吸合；在关闭的状态时 ,J1 端口输出低电平，继电器释放。 LED1.R2.D2 等元件构成的降压电路为模块提供 3V 工作电源， LED1 兼作电源指示， LED2 为开启状态指示。

五、操作方式

§ 键盘锁的操作

1 ．时钟显示状态下，长按 [ 取消 / 换路 ] 键超过 2 秒钟时会解除或启动键盘锁，上锁时 LCD 左下角显示“锁 ”符号，解锁后“锁 ”符号消失，如果 30 秒内不按任何键，系统会自动锁定键盘。

§ 时钟状态设定 / 调整

1. 只按 [ 时钟 ] 或 [ 小时 ] 或 [ 分钟 ] 或 [ 星期 ] 键时无作用。
2. 同时按 [ 时钟 ] 及 [ 小时 ] 键时可调整小时。
3. 同时按 [ 时钟 ] 及 [ 分钟 ] 键时可调整分。
4. 同时按 [ 时钟 ] 及 [ 星期 ] 键时可调整星期。
5. 每按一次 [ 开 / 自动 / 关 ] 键时会改变显示屏下方“”状态指示符的位置以及 J1 端口的输出状态：开 → 自动 → 关 → 自动 → 开循环；系统首次通电复位时， J1 端口输出默认为“自动”状态。
＊当此键按至“开”状态时：一直开
＊当此键按至“关 ” 状态时：一直关
＊当此键按到“自动 ” 状态时：自动执行已设定的时间程序

§ 定时状态设定 / 调整

1. 每按一次 [ 定时 ] 键时进入下一组定时，进入定时设置状态下：界面右下角显示定时的组数 , 共有 42 开 42 关可设置 < 连续按定时键 > ：
开 1 → 关 1 → 开 2 → 关 2 → 开 3 → 关 3 → …… → 开 42 → 关 42 循环；
2 ．按 [ 小时 ] 键时可调整各组定时之小时；
3 ．按 [ 分钟 ] 键时可调整各组定时之分钟；
4 ．按 [ 星期 ] 键时可调整各组定时之星期：
一 二 三 四 五 六 日 → 一 → 二 → 三 → 四 → 五 → 六 → 日 → 一 三 五→ 二 四 六→ 六 日 → 一 二 三→ 四 五 六→ 一 二 三 四 五 → 一 二 三 四 五 六 → 一 二 三 四 五 六 日 → …… 循环
5 ．按 [ 取消 / 恢复 ] 键，界面显示――：――，表示将此组定时取消使该组失去作用，再次按 [ 取消 / 恢复 ] 键可以恢复此组定时使其发生作用。
6 ．按 [ 时钟 ] 键时回到时钟状态，如果 30 秒内无任何键按下，系统会自动回到时钟状态。
